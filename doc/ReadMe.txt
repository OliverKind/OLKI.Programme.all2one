3 - Installations- und Deinstallationshinweise
**********************************************

a) Erstinstallation
-------------------
Wird die Software "all2one - File mover" das erste Mal, oder nach einer Deinstallation, installiert,
so wird ein Verzeichnis angelegt in dem die Benutzereinstellungen gespeichert werden.
Der Pfad zu diesem Verzeichnis ist üblicherweise (Bei Windows 7 oder späteren Versionen):
"C:\Users\[BENUTZERNAME]\AppData\Roaming\OLKI-Software\all2one"

Die Installation erfolgt für den zum Zeitpunkt der Installation angemeldeten Benutzer.
Weitere Informationen zu diesem Verzeichnis und den enthaltenen Dateien finden Sie in der Datei Readme.txt

b) Wiederholinstallation
------------------------
Wird die Software "all2one - File mover" installiert, ohne das vorher eine Deinstallation,
stattgefunden hat bleiben die Dateien im Einstellungsverzeichnis üblicherweise (Bei Windows 7 oder späteren
Versionen) "C:\Users\[BENUTZERNAME]\AppData\Roaming\OLKI-Software\ all2one " unbeeinflusst.
Sollte es erwünscht sein das Diese Dateien neu erzeugt werden so müssen sie zuvor vom Benutzer gelöscht
werden oder das Programm deinstalliert werden.

c) Deinstallation
-----------------
Bei der Deinstallation werden alle Programmdateien, Programmverzeichnisse und
die Einstellungsdateien und Einstellungsverzeichnisse gelöscht.
